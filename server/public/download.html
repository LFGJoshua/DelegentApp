Delegent Desktop

Track your work time, capture activity, and stay in sync with the dashboard — a lightweight desktop timer that runs quietly in the background.
⬇  Download
Windows · macOS
macOS is an unsigned build. First launch: right-click the app → Open → Open (or System Settings → Privacy & Security → Open Anyway).
If it says "damaged and can't be opened," open Terminal and run: xattr -dr com.apple.quarantine /Applications/Delegent.app
If Windows SmartScreen appears (unsigned app), click More info → Run anyway.
🔄 Windows installs once, then updates itself automatically
⏱️

Play / Pause timer

One click to start tracking. Pause anytime — your time only counts while you're working.

📊

Productivity light

A live activity light turns green when you're active 50%+, so you always know your focus level.

🔄

Auto-updates

The app checks for new versions on launch and installs them silently — no reinstalling, ever.

Getting started

1

Download & run the installer above. Delegent installs for your user (no admin needed) and adds a tray icon.

2

Open Delegent from the tray or Start menu, type what you're working on, and press ▶ Play.

3

Your time, screenshots, and activity appear on the team dashboard in real time.

4

Stay current automatically — when a new version ships, Delegent downloads and applies it on the next restart.
